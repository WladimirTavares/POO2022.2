Relação

Intro

Na matemática, uma relação é uma correspondência entre elementos de dois conjuntos não-vazios. Formalmente, se R uma relação definida do conjunto A para o conjunto B. O conjunto A é denominado conjunto de partidade e o conjunto B é denominado conjunto de chegada. Então R é qualquer subconjunto do A × B. Por exemplo, se A = {2, 3, 4} e B = {5, 6}.

  • {(2, 5), (3, 6)} é uma relação de A para B.

  • {(6, 3), (6, 4)} é uma relação de B para A.

Um tipo de relação importante é a relação em que o conjunto de partida e chegada são iguais.

Por exemplo, a relaçaõ ≤ no conjunto ℤ possui vários pares como (2, 2), (2, 3), (2, 4), …. A relação | (divide) no conjunto ℤ possui uma grande número de pares (2, 4), (4, 8), (3, 6), etc.

Neste exercício, vamos testar algumas propriedades das relações:

  • Reflexiva: Uma relação R em A é dita reflexiva quando para todo a ∈ A, (a, a) ∈ R.
  • Simétrica: Uma relação R em A é dita simétrica quando para todo a ∈ A para todo b ∈ A, (a, b) ∈ R, então (b, a) ∈ R. Observe que uma relação R = ∅ é simétrica, por vacuidade.
  • Transitiva:Uma relação R em A é dita transitiva quando para todo a ∈ A para todo b ∈ A, para todo c ∈ A (a, b) ∈ R e (b, c) ∈ R, então (a, c) ∈ R. Observe que uma relação R = ∅ é transitiva, por vacuidade.

A classe TestRelation cria uma instância da classe Relation. Em seguida, recebe mensagens e enviar essas mensagens para a instância da classe Relation.
